Fourier Transform Calculator

Overview

This project is a tool designed to compute the Fourier Transform of signals and functions. The Fourier Transform is a fundamental tool in signal processing and analysis, used to decompose a function into its constituent frequencies. This calculator was built as an academic exercise to demonstrate an understanding of the mathematical principles and their practical application in software.

My Role & Contributions

I was responsible for the core logic and implementation of the transformation algorithm. This involved translating the mathematical formula into a working program, handling user input for functions, and presenting the output in a clear format. My work focused on the backend computations to ensure accuracy and efficiency.
